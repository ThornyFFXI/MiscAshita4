Basic party buff and distance display.

Commands:
/pb distance - Toggle distance.
/pb exclusion default - Set to default exclusions, to match the tako version.
/pb exclusion [add/remove] [buff name] - Add or remove exclusions.
/pb priority [add/remove] [buff name] - Add or remove priority buffs.

Priority buffs will show leftmost before non-priority buffs.  Exclusions will not be shown at all
