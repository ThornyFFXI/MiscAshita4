Spellselect is an addon that allows you to cast spells by tier/element.

Commands(start with /ss or /spellselect):
/ss setele [fire/ice/wind/earth/lightning/water/light/dark] - Set element.
/ss setele any - Clear element requirement
/ss settier [t1/t2/t3/t4/t5/t6/ga/ga2/ga3/ja/am1/am2/eledot/helix/helix2/ra/ra2/ra3] - Set tier requirement.
/ss settier any - Clear tier requirement
/ss nuke [optional: ele/tier] [optional: ele/tier] - Cast the first available nuke matching set elements/tiers.  Specifications in command override settings.

Example:
/ss setele fire
/ss nuke t4
This will cast fire iv.

/ss setele fire
/ss nuke
This will cast the first available fire spell in the list.

/ss settier t4
/ss nuke
This will cast the first available tier 4 in the list.
