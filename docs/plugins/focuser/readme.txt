Focuser is a plugin written for multiboxers that allows you to change characters with typed commands.
Commands are as follows:
/winfocus [Character Name] - Swap to that character.
/winfocus *nextalpha - Swap to next character, alphabetically.  Rolls over.
/winfocus *prevalpha - Swap to previous character, alphabetically.  Rolls over.
/winfocus *nextpid - Swap to next character, by process id.  Rolls over.
/winfocus *prevpid - Swap to previous character, by process id.  Rolls over.
